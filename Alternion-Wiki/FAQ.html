FAQ

  • What exactly does your mod add?
    • SteamID customizable items
      • Individually selectable Weapon skins
      • Badge skins
      • Golden mask skins
      • Cannon skins (captains steamID)
      • Main Sail and Secondary Sail skins (captains steamID)
      • Flag skins (captains steamID)
    • Rotatable Main menu character
  • Do I have to manually install/update any textures when new things get added? No! The mod automatically fetches textures based off of what others and you have equipped at the time of starting up your game. Currently this means if anything is changed after you startup it won't show for you, so I plan to modify this in future if the mod gains enough traction for it to be noticable. All textures will be downloaded once the downloading indicator in the centre of your screen vanishes.

    If you wish to stop downloading the currently equipped items then you can disable the option called downloadOnStartup, which can be found on the third page of the config menu.
  • I have a feature request / Bug report, where do I go? Create an issue on the Github repo, and I will look at it as soon as I can, or alternatively contact me on discord Archie the inventor#4744.
  • Will other people see my loadout as well as me? For anyone else using the mod, Yes! All loadouts show for everyone on that server that also has the mod installed, as long as you didn't change your items after they started up their game. There is also support for updating users as they join their game, this however is currently still somewhat experimental.
  • Do I have to run the mod for others to see my items? Nope! Anyone using the mod will see your equipped items, whether you have it installed yourself or not!
  • How do I access the configuration menu? The default keybind is "]", it can be rebound in the AlternionConfig.cfg file, located in common\Blackwake\AlternionConfig.cfg. This file will also contain all the default settings with which you will startup each time. Any changes made ingame will not save to the config file UNLESS you click save in the config menu.
  • Can I get any of these features as individual mods? Currently the following are available as individual mods:
    • Rotatable Main Menu Character
    • Custom Flags
    • Custom Sails
